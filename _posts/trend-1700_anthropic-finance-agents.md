今天答案上架了。十个 ready-to-run 的金融服务 Agent 模板,作为 plugin 接进 Claude Cowork 和 Claude Code,另配一份 Managed Agents cookbook。

研究侧五个:pitch deck 生成器、客户会议准备器、财报回顾器、模型搭建器、行业研究员。运营侧五个:估值复核、总账对账、月末关账、报表审计、KYC 筛查。每个模板打包了三件东西——领域 skill、有权限管控的数据 connector、跑子任务的 subagent。Microsoft 365 那边 Excel、PowerPoint、Word add-in 已经 GA,Outlook 排队中。意思是做 banker 的不用切窗口,Claude 跟着上下文跳应用。

"那这东西有人在用吗,还是 PR 稿里挂个 logo?"

同框出现的八家:Citadel、BNY、Carlyle、Mizuho、Walleye、Hg、FIS、Travelers。最值得拎出来看的是 Walleye Capital——一家 400 人规模的对冲基金,Claude Code 内部 100% 采用率。不是 IT 部门统一推,是 400 个人都在用。一家对冲基金愿意把分析师、交易员、quant 全压在一个外部模型上,这个信号比 benchmark 数字重得多。

Anthropic 顺手公布了 Claude Opus 4.7 在 Vals AI 的 Finance Agent benchmark 拿 64.37%,行业第一。数据 connector 一排——Dun & Bradstreet、Moody's MCP、Verisk、Fiscal AI、FMP、Guidepoint、IBISWorld、SS&C IntraLinks、Third Bridge——基本把卖方研究员桌面要刷的库收齐了。

我看下来感觉是这样:上周的合资公司是策略,这周的十个模板是产品。Anthropic 跟 OpenAI"自己做垂直应用"的路线走的是反方向——它选的是"做基座 + 平台合伙人卖给行业"。金融是第一刀,法律和医疗大概率排在后面。
