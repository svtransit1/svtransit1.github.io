Anthropic Labs 刚上线 Claude Design——跑 Opus 4.7,吃文字、图、DOCX / PPTX / XLSX、你自己的代码库,还能抓网页元素;输出 HTML、PDF、PPTX、Canva,改完一键打包丢给 Claude Code 接着落地。Pro / Max / Team / Enterprise 订阅直接包含,用的是你原本的额度池。

真正狠的地方:它自动从代码库里读出设计系统,生成的东西默认就套你公司的规范。Datadog 的原话——"以前要一周的事,现在一场对话里就完了"。Brilliant 那组页面原本别的工具要 20+ prompt,这里只用 2 个。

上周 Opus 4.7 把视觉窗口放到 2576px,这周就接着落一个吃图的产品。Anthropic 在把 Claude 从写代码的工具,往上游的需求 / 产品 / 设计环节推。节奏咬得很紧。

这步棋打出来,Figma 那边要开始想怎么接招了。
