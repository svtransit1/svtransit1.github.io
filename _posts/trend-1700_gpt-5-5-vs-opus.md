今天整理五条能直接对齐的数字,来源是 OpenAI 的官方发布页 + 同期第三方报道,没有厂商幻灯片级别的"自报无对照"条目。

第一条——推理。GPT-5.5 Pro 在 FrontierMath Tier 4 上做到 39.6%,Opus 4.7 是 22.9%,差距接近 1.73 倍。这是今年以来 Anthropic 第一次在一条头部数学基准上被拉开这么多。注意这是 Pro 档成绩,默认的 GPT-5.5 本体并没有跑到这个分——OpenAI 自己的介绍页面也是两条分数分开列的,不要看到标题就代入全档。

第二条——代码。SWE-bench 88.7%。昨天 morphllm 那份 15 家代理横评里,Opus 4.5 是 80.9%,Claude Code 端到端已经接近天花板。现在 GPT-5.5 把上限直接抬了近 8 个百分点。但 Codex CLI 要把这条"裸模型"优势传到真正的 agent 工作流里,还得等 OpenAI 把 Agents SDK 那一套工具链铺完——现在还没。裸模型强,不等于代理栈强,这一层要分开看。

第三条——综合知识。MMLU 92.4%。这条数字本身意义越来越弱,因为头部模型都挤在 90+ 的天花板附近,差 2-3 个点已经很难说谁真的更"通识"。但它至少证明 GPT-5.5 没在刷数学的同时丢掉别的能力。

第四条——幻觉。OpenAI 自报比 GPT-5.4 降了 60%。这条只能先参考,独立复现一般要等两三周,三方评测才会出数据。内部测的幻觉降幅,历史上多少都会在现实使用里打个折扣。

第五条——分发。GPT-5.5 同步上线 Plus / Pro / Business / Enterprise,Codex 里直接能用,NVIDIA 的基础设施上也同步跑。从发布到 ChatGPT 用户刷到,零时差。Anthropic 这边 Opus 4.7 的 API + Claude Code 通道是快的,但消费端的流量池,比 OpenAI 还是差一个数量级——这不是谁好谁坏的问题,是两家公司的产品形态就不在同一个赛道上。

谁该用哪个:

· 做学术数学、奥赛题、长推理链、Frontier 级别的定量任务——现在 GPT-5.5 Pro 是硬优势,基本没悬念,换不了也不需要挣扎。

· 做长上下文代码代理、Claude Code / Agent SDK 那一套工具链——Opus 4.7 的 1M 上下文 + /ultrareview 的生态还没被追上,换过去成本不低,生态迁移的隐形成本会吃掉大部分裸模型差距。

· 做 C 端消费产品——GPT-5.5 的流量渠道 + Codex 入口更顺,这条几乎没得挑。

真正压脊椎的不是谁更聪明,是这个六周一代的节奏。任何一家公司停三个月,就自动被甩到上一代。前两年还以为 AGI 是一级级爬台阶,现在看更像赛马——一圈一圈换马,掉队就没人等你,团队和产品线都要按这个时钟重排。
